Amazon.com Clone

We wanted to recreate a complex online store.

As a group, we wanted to make a fully functional clone of Amazon. We only had two weeks to make something that has taken several teams years to build, so we had to choose wisely which features we would be recreating. We focused on the main functions including, creating an account, logging in, sorting filters, and of course their sleek design.

Paul-Valentine.com Clone

After seeing Paul-Valentine.com win an award for it's sleek design, I decided to clone their site.

Paul Valentine Clone is a clone of an E-Commerce site for wrist watches. My cloned version includes a home view, product view, a shopping cart, and a checkout using Stripe the API. This was all built in AngularJS, Node/Express, and SQL
